## USAGE
```
nodebrew install-binary latest  
nodebrew use v13.10.1  
echo 'export PATH=$PATH:$HOME/.nodebrew/current/bin' >> ~/.bashrc && source ~/.bashrc  
sudo npm -d install electron-prebuilt 
```
